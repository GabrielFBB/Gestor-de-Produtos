Gestor de Produtos | Python, Tkinter, SQlite
Aplicação desktop para gestão de produtos com interface gráfica. Permite cadastrar, editar, eliminar e listar produtos
com nome e preço, com os dados persistidos em banco de dados local. Elimina o controlo manual de produtos em
papel ou planilhas simples, oferecendo uma interface visual intuitiva para gerir um catálogo de produtos de forma
rápida e organizada.
